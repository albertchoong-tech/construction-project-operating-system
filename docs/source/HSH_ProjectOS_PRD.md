# HSH ProjectOS PRD

## Project Name

**HSH ProjectOS -- Integrated Construction Project & Financial
Management Platform**

## Vision

Build a Construction Project Operating System (ProjectOS) that manages
the complete lifecycle of residential construction projects---from
quotation and budgeting to procurement, site progress, variation orders,
progress claims, cash collection, supplier payments, and project
profitability---in one integrated platform replacing Excel, WhatsApp,
and manual paperwork.

## Primary End-to-End Workflow

1.  Create quotation.
2.  Convert approved quotation into a project.
3.  Create project budget and BOQ.
4.  Raise purchase requests.
5.  Approve purchase orders.
6.  Record deliveries and supplier invoices.
7.  Update daily site progress.
8.  Raise variation orders when required.
9.  Generate progress claims based on work completed.
10. Record customer receipts.
11. Record supplier payments.
12. Automatically update project cash flow and profitability dashboard.

## Success Criteria

Within one week, one live construction project can be managed entirely
inside the system without relying on Excel or WhatsApp. Every
stakeholder sees the same real-time project status, financial position,
pending approvals, and project profitability.

## User Roles

### Director

Full access to dashboards and approvals.

### Project Manager

Manage projects, progress, budgets and resources.

### Quantity Surveyor

Manage BOQ, quotations, variation orders and claims.

### Purchasing Officer

Manage procurement, suppliers and purchase orders.

### Finance

Manage receipts, payments, cash flow and profitability.

### Site Supervisor

Submit daily progress, photos, inspections and issues.
